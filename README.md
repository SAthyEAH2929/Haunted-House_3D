# 🏚️ Haunted House 3D

A 3D stealth game developed using the Unity Engine as part of the Unity Learn **3D Stealth: Haunted House** project.

## 🎮 About the Game

Haunted House 3D is a stealth-based game where the player explores a haunted mansion while avoiding enemy guards. The objective is to navigate through the environment, stay out of enemies' sight, and reach the exit without being detected.

The project demonstrates core Unity game development concepts including player movement, enemy AI, navigation, lighting, animations, and game interactions.

---

## ✨ Features

- 🎮 Third-person player movement
- 👁️ Enemy AI with field of view detection
- 🚶 NavMesh pathfinding
- 💡 Dynamic lighting and shadows
- 🏠 Haunted house environment
- 🔊 Background audio and sound effects
- 🚪 Interactive gameplay
- 🖥️ Windows standalone build
- 🌐 WebGL build included

## 🚀 How to Play

1. Clone this repository.

```
git clone https://github.com/SAthyEAH2929/Haunted-House_3D.git
```

2. Open the project (or run the executable).

3. Launch:

```
Haunted House.exe
```

4. Complete the level without getting caught by the enemies.
